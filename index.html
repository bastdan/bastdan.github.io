Four terms you need to know

Before we dive in

01

Coworker — A persistent, goal-driven multi-agent orchestrator that monitors signals, decides next-best actions, and executes across channels. Example: CX Enterprise Coworker (NVIDIA-co-built runtime). Positioned above individual agents.

02

Agent — A single-purpose specialist invoked for a bounded task. Examples: Journey Agent (AJO), Workflow Optimization Agent (Workfront), Site Optimization Agent, Migration Assessment Agent. Can be composed by a Coworker.

03

Skill — A reusable instruction package (prompt + tool bindings + constraints) composable into agents. Published in the Agent Skills Catalog. Lets VILT package expertise as callable units.

04

MCP Server — Anthropic's Model Context Protocol. Adobe products (AEP, Commerce, Marketo, JO, AEM) now expose native MCP servers so external agents — Claude, ChatGPT, Copilot, Gemini — can securely operate on Adobe data/actions.

Presenter: Without this slide, the rest of the deck is a word salad — every feature references at least one of these terms. The important taxonomy insight: Coworker ≠ Agent ≠ Skill ≠ MCP.
